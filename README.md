## Setup

### Prerequisites

- Ruby (version 3.2.2)
- Rails (version 7.0.5)
- PostgreSQL

1. Clone the repository:
    ```sh
    git clone https://github.com/rudzainy/ranting.git
    cd ranting
    ```

2. Install dependencies:
    ```sh
    yarn
    bundle install
    ```

3. Set up the database:
    ```sh
    bin/rails db:create
    bin/rails db:migrate
    ```

## Run

Start the development server:
```sh
bin/dev
```
